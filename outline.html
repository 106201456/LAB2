Unit Outline

COS10026 Web Technology Project

Semester 1, 2025

Aims

This unit is about learning how to build websites and web apps. We will do projects, work in teams, and learn about HTML, CSS, JS, databases and security. The goal is to get practical skills to make working web applications.

Unit Learning Outcomes

  1. Understand basic web tech and components.
  2. Make a project plan and manage tasks.
  3. Work and communicate in a team.
  4. Use databases (create, update, delete etc.).
  5. Build simple web apps with ethics/security in mind.
  6. Know simple security ideas and apply them.
  7. Think about ethics, privacy and accessibility.

Content

  • Front-end and back-end basics
  • Project planning + Git
  • Databases and SQL
  • Security and ethics
  • Accessibility
